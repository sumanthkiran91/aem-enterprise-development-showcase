# AEM Component Development

Generic AEM component examples using HTL, Sling Models, Dialog XML, ClientLibs, and content structure.

## Included Components
- Page Title Component
- Sub Heading Component
- Tabs Component
- Grid Component
- CTA Button Component
- Dynamic Table Component

```mermaid
flowchart TD
    A[Requirement] --> B[Dialog Design]
    B --> C[HTL Markup]
    C --> D[Sling Model]
    D --> E[ClientLib CSS / JS]
    E --> F[Authoring Test]
    F --> G[Publish Test]
    G --> H[Accessibility Check]
    H --> I[Code Review]
```
