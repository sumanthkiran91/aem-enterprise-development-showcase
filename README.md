# Enterprise Customer Journey

A generic multi-step enterprise journey that can apply to multiple industries.

## Journey Steps
1. Customer Details
2. Product / Service Selection
3. Additional Information
4. Review Details
5. Confirmation

```mermaid
flowchart LR
    A[Customer Details] --> B[Product / Service Selection]
    B --> C[Additional Information]
    C --> D[Review Details]
    D --> E[Confirmation]
```
